# Qt Widgets Inventory (Theming Reference)

> **Canonical document (Chinese)**：[../zh/qt-widgets-inventory.md](../zh/qt-widgets-inventory.md)

Inventory of Qt 6 Widgets (and adjacent modules) for **QThemeEngine** coverage planning.

- Implementation status lives in [../zh/coverage-matrix.md](../zh/coverage-matrix.md).  
- Product path: ThemeStore + `QThemeStyle` (not QSS) — see [architecture.md](architecture.md).  
- Upstream: [Qt Widget Classes](https://doc.qt.io/qt-6/widget-classes.html).

The Chinese inventory lists classes by family, tags theme priority (**T0–T3**), and marks module boundaries (`OpenGLWidgets`, `PrintSupport`, etc.).
